# My CV & portfolio

Welcome to the repository for my personal web-based CV and portfolio, hosted via **GitHub Pages** at:

**https://richardsgz.github.io**

The website showcases my academic background, professional experience and skills, and will soon include a few showcase projects.  

---

## About the website

The site is designed to be lightweight and easy to maintain, built as a simple static website using HTML, CSS and JavaScript.

It currently includes:

- A web-based version of my CV
- Contact details.

It will soon also include:

- Main publications
- Research interests
- Showcase projects

---

## Viewing the site locally

To preview before pushing updates:

```
# From inside the repo
open index.html		# macOS
# OR
xdg-open index.html	# Linux
# OR
start index.html	# Windows
```
